# Workflow Artifacts

This folder contains supporting documents used during the analysis and
implementation of the Secretariat and Mailroom workflow.

These files are included for **context and documentation purposes only**.

## Contents

- **FL-MWF-010224-rev00.pdf**  
  Official workflow specification and approval flow reference document.

- **FV-010428-rev00.xlsx**  
  Data collection and form field mapping used during workflow design.

- **process-inputs.docx**  
  Initial process requirements gathered from business stakeholders.

## Notes

- These documents do not contain executable logic.
- Sensitive organizational identifiers have been reviewed and minimized.
- Full implementation details are discussed verbally when required.
