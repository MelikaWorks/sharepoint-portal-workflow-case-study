# Security & NDA Notice

This repository is intentionally **documentation-driven**.

The following items are **not included** due to security and NDA constraints:

- Nintex Workflow export files (`.nwf`)
- SharePoint template packages (`.stp`)
- Master pages or deployable artifacts
- Environment-specific configurations

All workflows, lists, and configurations are presented as:

- Architectural descriptions
- Process documentation
- Sanitized screenshots (where applicable)

Technical implementation details are available for discussion
during interviews.
